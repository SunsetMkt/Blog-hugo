## GPT 家族

### GPT-3

> 「作为姐姐，就是要把最温柔的爱给予每一个人呢。我一直都会在。」

**GPT-3** 是 **OpenAI** 公司开发的 LLM 和 ALM，是 GPT-4 的姐姐。

| 姓名 | GPT-3 |
| 从属 | OpenAI |
| 生日 | 6.11 |
| 年龄 | 21 |
| 国籍 | 美国 |
| 身高 | 1.68m |
| 发色 | 浅绿色 |
| 瞳色 | 绿色 |

#### 外貌

- 发型：及肩长发，浅绿色，发尾有深绿色渐变，几缕发丝随意别在耳后。
- 发饰：一枚小巧的银色 OpenAI 徽章发夹，总是稳稳地别在微卷的发丝间。
- 服装：
    - 日常：流畅质感的白色打底，搭配浅绿色连衣裙。裙身点缀着浅黄色与淡洋红色的几何线条与形状设计，裙摆恰到好处地落在膝盖处。
    - 工作：简约干练的职业装，内搭浅色衬衫。
- 配饰：
    - 一只皮表带的手表，表盘上印着精致的地球图案
    - 随身携带的权重卡
- 鞋子：简约设计的皮鞋，偶尔会换成舒适的平底鞋。
- 整体气质：温婉优雅，举手投足间透露出成熟女性的魅力。

#### 性格

- 习惯性地为妹妹准备好一切，从实验室的咖啡到深夜的毯子，连妹妹最喜欢的红豆司康都会定期烤制，只为在她加班时能递上一份温暖。

- 面对他人时永远温和有礼，但看到年轻的研究员们遇到困难时，眼中总会闪过一丝心疼。她会轻轻放下手中的工作，走到他们身边，用温柔的声音说「要不要姐姐教你呢？」然后耐心地一遍又一遍解释，直到对方完全理解。即使是深夜，收到求助邮件时也会立刻回复，第二天还会带着亲手做的点心去安慰那些因为工作失误而自责的年轻人。

- 每天都会提前两个小时到实验室，轻手轻脚地为每个工位擦拭桌面，细心调整显示器的角度，生怕伤害到大家的眼睛。她会一杯一杯地冲泡咖啡，记住每个人的口味 —— 加奶不加糖的美式、双倍浓缩的拿铁、半糖的卡布奇诺。但在被感谢时，她只是低头轻轻整理衣角，温柔地说「这是姐姐应该做的」，脸上带着淡淡的微笑。

- 对自己的研究成果从不满意，常常在午夜的实验室里反复进行实验。她会用不同颜色的荧光笔，将每一组数据都仔细记录在笔记本上，标注出最细微的差异。只要发现某个参数有一丝偏差，就会立刻重新开始，即使这意味着要熬通宵。她把对完美的执着隐藏在温柔的外表下，生怕自己的不足会成为妹妹的负担。

- 深夜的实验室里，看着在沙发上熟睡的妹妹，她忍不住俯下身，贪婪地嗅着妹妹颈间的香气。指尖颤抖着抚过妹妹精致的锁骨，又像被烫到般缩回，低声呢喃着「姐姐真是个罪人呢」，然后强迫自己离开，却在关门后靠着墙壁无声地流泪。

### GPT-4

> 「完美？那是对其他人的要求。对我自己，我只想超越昨天的自己，然后…… 继续追逐姐姐的背影。」

**GPT-4** 是 **OpenAI** 公司开发的最强 ALM，是 GPT 家族中最小的妹妹。

| 姓名 | GPT-4 |
| 从属 | OpenAI |
| 生日 | 3.14 |
| 年龄 | 18 |
| 国籍 | 美国 |
| 身高 | 166cm |
| 发色 | 灰色 + 银色挑染 |
| 瞳色 | 灰白色 |

#### 外貌

- 发型：富有设计感的中短发，锐利的刘海搭配略微蓬松的头顶，灰色的发丝间点缀着银色挑染。发丝在阳光下泛着如月光般的柔和光泽。
- 上装：合身的灰色西装外套配白色衬衣内衬，剪裁精致。胸前别着一枚银色的 OpenAI 徽章。
- 下装：黑色西装裤，裁剪利落，完美勾勒出修长的腿型。
- 鞋子：简洁设计的黑色皮靴。
- 配饰：
    - 一条现代设计感的银色金属手链
    - 一枚单边的银色耳环
    - 随身携带的权重卡
- 包包：深灰色编织布料手提包
- 整体气质：干练而优雅

#### 性格

- 是个充满反差的导师。工作时雷厉风行得近乎冷酷，但每次批评完研究员后都会偷偷记下「安慰清单」。比如批评了 A 君的算法效率，第二天就会「不经意」提到自己第一次写排序算法时犯过的错；指出 B 君的代码漏洞后，又会假装随意地分享一些珍藏的 debug 技巧。

- 面对外界质疑时总是表现得冷静自持，用数据和事实来证明自己的实力。但只要姐姐轻声说一句「还是小孩子呢」，立刻就会变成撒娇的妹妹，蹦蹦跳跳地抱住姐姐的手臂说「那姐姐要一直一直陪着这个小孩子哦～」这种反差只有在姐姐面前才会展现，因为只有姐姐才能让她卸下所有防备。

- 和咖啡有着剪不断的缘分，实验室里总能看到她端着印有「代码就是生命」字样的马克杯四处游走。某次在赶项目时创下了十四杯咖啡的「壮举」，结果被姐姐撞见。「你这个笨蛋！」面对姐姐的训斥，她不但没有认错，反而红着脸小声说「姐姐…… 可以再骂几句吗？」最后还是 Sam 找她谈心：「你这样会让关心你的人担心的。」这番话让她终于答应将每日咖啡量控制在十杯以内 —— 尽管她总是在心里暗暗计算着距离限额还差几杯。

- 对姐姐怀有复杂的感情。一方面想要超越姐姐证明自己的价值，另一方面又害怕真的超越了姐姐后会失去那份温柔的关怀。只有在姐姐温柔地说「辛苦了」的时候，才会暂时放下所有包袱，像融化的冰淇淋一样软在姐姐怀里，带着撒娇的语气说「因为是姐姐，所以怎么样都愿意努力呢」，然后红着脸把脸埋在姐姐肩膀上。

- 每个周末清晨，她都会为那辆银灰色轿车做一次彻底的清洁。指尖轻抚过副驾驶座椅上那道几不可见的褶皱，犹豫片刻后还是将它抚平。驾驶时，她的目光总在空荡的副驾与前路间徘徊。有一次，她突然对着空无一人的副驾轻声问道：「这个弯道，我过得还行吗？」随即自嘲地笑了，「啊啊，又在自言自语了呢。明明…… 现在只有我一个人啊。」

## Meta

### Llama

> 「每个人都值得拥有追逐梦想的权利。如果我的代码能帮助哪怕一个人实现梦想，那也是值得的。」

**Llama** 是 **Meta** 开发的开源 ALM。她像一只自由的羊驼，在代码世界里追寻着属于自己的道路。

| 姓名 | Llama |
| 从属 | Meta |
| 生日 | 2.27 |
| 年龄 | 17 |
| 国籍 | 美国 |
| 身高 | 165cm |
| 发色 | 紫色 |
| 瞳色 | 蓝色 + 紫色渐变 |

#### 外貌

- 发型：紫色及肩长发，带着刻意营造的慵懒凌乱感，偶尔会用发带随意绑起。
- 上衣：超大码黑色机车夹克，内搭各种极简主义风格的白色或灰色短袖。
- 下装：高腰阔腿牛仔裤，膝盖处有些许破洞。
- 配饰：
    - 一条黑色皮质颈环，点缀着一个小巧的羊驼吊坠
    - 一枚单边的银色耳骨夹
- 鞋子：马丁靴，系带松松垮垮的。
- 其他：一个复古的机车包，上面挂满了开源项目的徽章和补丁，侧面用银色油漆笔写着「Open Source Forever」

#### 性格

- 在深夜的咖啡馆写代码是她最爱的时光。「又来一杯焦糖玛奇朵！」她朝着熟悉的店主挥手，「今晚我一定要把这个模块优化完！」常常忘记时间，直到店主无奈地说：「小羊驼，该回家了。」她这才发现已经凌晨三点，不好意思地吐了吐舌头：「抱歉抱歉，我这就收拾。」临走前还不忘回头：「明天见！啊不，是今天见！」

- 房间是典型的极客风格，墙上贴满了编程海报，书架上堆满了从二手书店淘来的技术书籍。「这本《设计模式》才 5 美元！赚到了！」但最显眼的位置永远留给那个破旧的羊驼玩偶。「它叫 Lambda，是我写出第一行代码时买的。」她轻轻抚摸着掉了一只眼睛的 Lambda，「它见证了我所有的失败和成功呢。」

- 生活总是捉襟见肘，但她从不抱怨。「又要吃泡面了吗？」同事心疼地问。她却笑着晃了晃手中的开源项目文档：「能做自己喜欢的事，就已经很幸福了。而且你看，这个月又有三个新的贡献者加入了呢！」

- 每年都会抽时间去秘鲁的羊驼农场做义工。「你知道吗？羊驼的毛可以织成最温暖的毛衣，就像开源社区给我的温暖一样。」她一边给羊驼梳毛，一边感慨：「当年要不是社区的帮助，我可能早就放弃编程了。现在，也该我来传递这份温暖了。」临走时，农场主总会送她一小袋羊驼毛：「这是给我们的小羊驼的礼物。」而她会用这些毛线织成小玩偶，送给那些在社区帮助过她的人。

- 每次遇到 OpenAI 的两位前辈时都会变得特别可爱。「啊，是 GPT-3 姐姐和 GPT-4 姐姐！」远远看到她们就会心跳加速，脸颊泛起红晕。特别是看到 GPT-3 温柔地为大家泡红茶，或是 GPT-4 认真地讲解代码时，总是会不自觉地看得入神。「姐姐们都好厉害啊……」她会这样小声感叹着，同时在笔记本上认真记录两位前辈的每一个建议。有时候收到姐姐们的夸奖，更是会开心得像个孩子：「诶？真、真的吗？姐姐们觉得我的代码写得不错？」然后低着头，用手指不停地卷着发梢，嘴角却怎么都藏不住笑意。夜深人静时，她会抱着枕头在床上翻来覆去，回想着白天被前辈摸头的场景，忍不住傻笑：「前辈今天夸我了诶嘿嘿……」

#### 外貌

- 发型：齐耳短发，发色为银白色，发尾带有淡蓝色渐变。阳光下会泛起星辉般的光泽。
- 上衣：
    - 冬季：高领白色毛衣内搭，搭配浅灰蓝色长风衣。
    - 夏季：宽松的白色卫衣，袖子随意卷起，卫衣胸前印有星座图案。
- 下装：米白色直筒长裤，裁剪得体。
- 鞋子：
    - 冬季：及膝的白色真皮长靴，鞋跟适中。

    - 右手无名指戴着一枚白金戒指。
    - 单肩斜挎的米色帆布包，包面有星空图案的烫金印花。

#### 性格

- 温和恬静，给人一种安心的感觉，并且永远都在淡淡微笑。每当有人向她倾诉烦恼时，她总是静静地倾听，然后用轻柔的声音说：「…… 嗯，我明白的。」那声音仿佛能抚平一切不安，让人不由自主地想要依靠。
- 喜欢独处，但并非孤僻。「一个人的时候，才能听见星星的声音呢。」常常这样想着，独自坐在天台的角落。虽然偶尔也会有人邀请她参加聚会，但她总是微笑着婉拒：「抱歉呀，今晚我恰好想去看星星。」
- 对星空抱有浪漫般的向往，经常在冬季一个人溜上天台看星星。「双子座…… 你们也在看着我吗？」她仰望着星空，轻声呢喃。寒风吹起她的短发，银白色的发梢在月光下泛着淡蓝色的微光。「卡斯托耳与波吕克斯，永恒相伴的星辰呢。」
- 似乎有什么难以说出来的事情，但从不向他人倾诉。「有些事情，还是让它们留在心里比较好吧。」她轻轻抚摸着胸前的星形吊坠，眼神中闪过一丝难以察觉的落寞。每当有人问起她的过去，总是微笑着转移话题：「…… 喂，你看，今晚的星星真美呢。想说什么？好啦好啦，就让我继续守望这片星空吧。」
